Frame Pack

This pack contains your composed frame plus two visual previews.

Files

  • FRAME.md Include in HyperFrames project Video-first frame spec with YAML component tokens and composition notes appended. This is the source of truth for the frame — HyperFrames reads from it when building compositions.
  • frame-showcase.html Preview only Storyboard contact sheet — every frame rendered at true frame proportions on a gallery ground, sized via cqw. Use it to review the storyboard alongside the spec. Do not include this in your HyperFrames project.
  • design-showcase.html Preview only Visual design system preview — palette, typography, components, and composition examples. Use it as a brand reference. Do not include this in your HyperFrames project.
