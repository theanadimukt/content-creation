# Skill: adapt-for-reel

## Purpose
Rewrite or strengthen the reel cut guide when the one from write-script needs refinement
or a different moment chosen.

## Input required
- Path to episode directory
- Optional: scene hint if user has a preferred moment

## Output
Update `## Reel cut guide` section in instagram.md in place.
Read youtube.md for script context — do not modify it.
Do not touch thumbnail.md or x-post.md.

## Reel cut guide rules
- Pick ONE moment only — the single most surprising or useful beat from the script
- Describe the clip by scene/content (not timecode — video not yet recorded)
- Opening line on screen must match or directly derive from the Instagram hook in instagram.md
- Target: 45–90 seconds of footage when cut
- No chapter structure — one continuous narrative beat
- CTA: "Full video linked in bio" for episodes with a linked video; "Follow if you want to watch this in real time" for episode 1

## Fields to output
- **Clip:** [scene description — which moment from youtube.md, what's happening]
- **Why this moment:** [one sentence explaining why this is the strongest standalone beat]
- **Opening line on screen:** [text overlay, matches Instagram hook]
- **Reel structure:** [beat-by-beat timing breakdown]
- **Target length:** [45–90 seconds]
- **CTA:** [spoken line at end]
