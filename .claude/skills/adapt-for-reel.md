# Skill: adapt-for-reel

## Purpose
Rewrite or strengthen the ## Reel cut guide section in an existing script.md.
Use when the cut guide generated by write-script needs refinement or a different moment chosen.

## Input required
- Path to episode's script.md
- Optional: scene hint if user has a preferred moment

## Output
Update the ## Reel cut guide section in the existing script.md in place.
Do not touch any other section.

## Reel cut guide rules
- Pick ONE moment only — the single most surprising or useful beat from the script
- Describe the clip by scene/content (not timecode — video not yet recorded)
- Opening line on screen must match or directly derive from the Instagram hook
- CTA is always: "Full video linked in bio"
- Target: 60-90 seconds of footage when cut
- No chapter structure — one continuous narrative beat

## Fields to output
- **Clip:** [scene description — which moment, what's happening]
- **Why this moment:** [one sentence explaining why this is the strongest beat]
- **Opening line on screen:** [text overlay, matches Instagram hook]
- **CTA:** "Full video linked in bio"
